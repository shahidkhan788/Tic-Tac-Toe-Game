# Tic-Tac-Toe: Player vs Computer

A simple command-line Tic-Tac-Toe game developed in Python. The player competes against a computer opponent on a 3×3 board. The game updates the board after each move, prevents duplicate selections, and automatically determines winners or draws based on standard Tic-Tac-Toe rules.

## Features

* Player vs Computer gameplay
* Interactive command-line interface
* Dynamic board updates
* Input validation
* Win detection
* Draw detection
* Randomized computer moves

## Technologies Used

* Python 3
* Random module

## How to Play

1. Run the Python script.
2. Enter a number from 1 to 9 corresponding to a position on the board.
3. The computer will automatically make its move.
4. Continue until either the player wins, the computer wins, or the game ends in a draw.
